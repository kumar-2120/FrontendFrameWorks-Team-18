### How to Use

1. **Open the Application**: Simply open `index.html` in a web browser
2. **Select Your Raw Product**: Choose from milk, fruits, vegetables, grains, honey, or herbs
3. **Choose Conversion Type**: Select the value-added product you want to create
4. **Enter Quantity**: Input how much raw product you have
5. **View Results**: See the expected output, revenue increase, processing time, and equipment needed

### Installation & Setup

No installation required! This is a static web application that runs directly in your browser.

#### Running Locally

1. Clone this repository:
   ```bash
   git clone https://github.com/kumar-2120/FrontendFrameWorks-Team-18.git
   ```

2. Navigate to the directory:
   ```bash
   cd FrontendFrameWorks-Team-18
   ```

3. Open `index.html` in your web browser:
   - Double-click the file, or
   - Use a local server (recommended):
     ```bash
     python -m http.server 8000
     ```
     Then visit `http://localhost:8000`
